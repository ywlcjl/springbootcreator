# SpringbootCreator

是一款由Springboot编写的通用CMS后台管理系统, 该系统能够让您快速搭建一个Java Web的Api应用, 各类需要登录后台的账号管理基础系统, 以及CMS文章资讯类网站系统等应用.

# 框架功能与技术结构如下:

# 后端技术:
* Springboot, MySQL8.0, Mybatis-plus, Freemarker

# 前端技术:
* Bootstrap5, JQuery3, Summernote editor

# 核心功能:
* 用户管理, 权限管理, 文章管理, 分类管理, 图片管理, 系统日志等
